Proyek ini dibuat untuk memenuhi tugas **Praktikum Laravel Pertemuan 6**.  
Aplikasi yang dikembangkan bernama **SMP Mentari**, yaitu website sekolah berbasis **Laravel 12** dengan integrasi **Filament v4** sebagai panel admin.

- Mengintegrasikan **Filament Panel**,
- Membuat fitur **CRUD (Create, Read, Update, Delete)** untuk dua entitas:
  - **Kegiatan Sekolah**
  - **Siswa**
- Menghasilkan tampilan yang dapat digunakan oleh pengguna/admin sekolah.
